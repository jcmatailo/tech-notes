# Keyboard shortcuts

| **Shortcut** | **Description** |
|-------------|-----------------|
| Ctrl + A | Move to the start of the line |
| Ctrl + E | Move to the end of the line |
| Ctrl + U | Delete from the cursor to the start of the line |
| Ctrl + K | Delete from the cursor to the end of the line | 
| Ctrl + C | Stop the current process/command | 
| Ctrl + D | Log out or exit the terminal |
| Ctrl + L | Clear the terminal screen |
| Ctrl + R | Search command history |
| Ctrl + W | Erase one word in the current line |
| Ctrl + Y | Paste the last delete word |
| Ctrl + Z | Suspend the current foreground process |
| Ctrl + T | Transpose the character before the cursor |
| Ctrl + H | Delete the character before the cursor (Backspace) |
| Ctrl + B | Move backward one character |
| Ctrl + F | Move forward one character |
| ALT + B | Move backward one word |
| ALT + F | Move forward one word |
| ALT + D | Delete the word after the cursor  |
| Alt + . | Insert the last argument from the previous command  |
| Tab	| Autocomplete commands, files, or directories |
| Tab (twice) | List all possible completions |
